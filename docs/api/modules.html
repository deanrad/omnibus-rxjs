after

  • after<T>(ms: number, objOrFn?: T | Thunk<T> | Observable<T>): AwaitableObservable<T>
    • Defined in src/after.ts:18

    after is a composable setTimeout, based on Observables. after returns an Observable of the value, or result of the function call, after the number of milliseconds given. For a delay of 0, the function is executed synchronously when .subscribe() is called. after is 'thenable' - and can be awaited like a Promise. However, since underneath it is an Observable, after is both lazy and cancelable!
